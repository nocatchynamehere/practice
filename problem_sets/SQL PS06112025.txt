Task:
Write a query to return each userâ€™s name and their total login count, but only include users who logged in more than once.
Output columns: name, login_count

SELECT
  u.name,
  COUNT(*) as login_count
FROM
  `users` u
JOIN
  `logins` l
ON
  u.user_id = l.user_id
GROUP BY
  u.name
HAVING
  COUNT(*) > 1
